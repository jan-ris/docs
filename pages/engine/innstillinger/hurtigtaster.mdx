## Nyttige Hurtigtaster

| Tast | Kommando | Beskrivelse |
| ---- | -------- | ----------- |
| Ctrl + C | Kopier          | Kopier møbelelementer.  |
| Ctrl + V | Lim inn         | Lim inn møbelelementer. |
| Ctrl + Z | Angre           | Angre siste handling.   |
| Delete   | Slett           | Slett elementer.        |
| Escape   | Avbryt          | Avbryt handling eller fjern markering av elementer eller områder. |
| Shift    | Multi-seleksjon | Marker flere elementer eller områder samtidig. |
